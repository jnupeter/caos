Style Guide

Layouts

The #wrap div is used for the responsive menu.

The #header div includes the banner. This is mainly due to the positioning of the down-arrow for indicating the current main menu selected.

The .container class is used throughout the markup to center the divs, i.e. margin: 0 auto.

There are two logos, one for normal and one for tablet to mobile responsive.

The rest of the header is pretty static apart from user details.

Menu

Apply the class .current to the active menu item.

<ul id="menu"> <li class="first current"><a href="#" class="current">Home</a></li> <li><a href="#">Messages</a></li> <li><a href="#">My Account</a></li> <li class="last"><a href="#">Tests &amp; Workshops</a></li> </ul>

For pages 4 and 5, the menu has 5 items, so you need to add the class five to the #menu div.

<ul id="menu" class="five"> <li class="first"><a href="#" class="current">Home</a></li> <li><a href="#">Messages</a></li> <li><a href="#">My Account</a></li> <li class="current"><a href="#">My Application</a></li> <li class="last"><a href="#">Tests &amp; Workshops</a></li> </ul>

Banner

This is the standard wrapper for the banner div.

<div id="banner"> <div class="dots"> <div class="container"> ... </div> </div> </div>

Within the .container div you can use the following options:

<div class="banner-left"> ... </div> <div id="key-dates"> ... </div>

.banner-full is usually used for sub pages (i.e. pages 4 and 5) with a heading 1.

<div class="banner-full"> <h1>Heading 1</h1> </div>

Boxes

Boxes must be contained within a div with the col-1, col-2, col-3, col-4 and an inner div with the classes grey ui-corner-tl fixed-height.

.fixed-height is optional and will be ignored in tablet and mobile responsive.
